lib
===

User and kernel space libraries.

Put here the CMSIS (and other) libraries provided by the chip manufacturers.

*Recommended directory structure:*

    .
    |-crt
    |--armv6-m-libgcc
    |--libaebi-cortexm0
    |--rpi-libgcc
    |-Discovery
    |-STM32F0xx
    |--CMSIS
    |--Drivers
    |---inc
    |---src
    |--Discovery
    |-STR912
    |--stdlib
    |---inc
    |---src
